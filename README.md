# Animated Godot Button
An animated button for Godot 4.2.1.

![Gif of the Animated Button](https://github.com/artmouse155/godot_animated_button/blob/main/Demo%20Pics/button.gif?raw=true)

This animated button has high customability, with multiple adjustable parameters.

![Multiple Parameters](https://github.com/artmouse155/godot_animated_button/blob/main/Demo%20Pics/Screenshot%202024-05-18%20112711.png?raw=true)

# How to use
Place the "Animated Button" folder in your Godot project, and then drag the animated_button.tscn file into the scene you want an animated button. Adjust the parameters, such as text, color, and font, to your liking.

To use the functionality of the button, connect its "pressed" signal to something as you would use a regular Godot button.
